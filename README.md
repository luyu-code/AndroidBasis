# AndroidBasis
(持续更新中)企业级 Android 开发基础框架，克隆下来本代码后，即可快速开发 Android 项目。该框架提供了常见的功能模块，包括网络请求、数据存储、UI 组件等，帮助开发者快速搭建项目基础架构。

## 功能特性
- **网络请求**：集成 Net库[Net](https://github.com/liangjingkanji/Net)，Android 基于协程/OkHttp网络请求工具
